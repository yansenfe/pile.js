import Dialog from 'pile/dist/components/dialog'
const  {Alert} = Dialog

<Alert
  show={true}
  type="success"
  title="我是pile"
  buttons={[
    {
      label: '好的',
      onClick: function(){}
    }
  ]}
/>

属性

参数 描述 数据类型 默认值
show是否展示警告框bool
type类型，错误：warning，成功：successstringwarning
title警告框文字string
buttons按钮数组array
buttons.label标签名称string
buttons.onClick点击事件function
MaskClick点击蒙板回调function
width自定义宽度默认267pxstring
